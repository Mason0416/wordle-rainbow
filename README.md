# 🌈 Wordle Rainbow (Mobile Ready)

A modern Wordle-inspired web game with a colorful twist.  
Built with React, featuring smooth animations, a large word dataset, and full mobile keyboard support.

---

## 🎮 Demo

👉 https://Mason0416.github.io/wordle-rainbow/

---

## ✨ Features

- 🔤 5-letter Word Guessing Game
- 📚 Large Word Dataset
  - Powered by an-array-of-english-words
- 🎨 Rainbow Theme by Day
  - Monday → Red  
  - Tuesday → Orange  
  - Wednesday → Yellow  
  - Thursday → Green  
  - Friday → Blue  
  - Saturday → Indigo  
  - Sunday → Purple  
- 📱 Mobile-Friendly
  - Tap anywhere to bring up the keyboard
  - Fully playable on phone
- ⌨️ Dual Input Support
  - Physical keyboard (desktop)
  - On-screen keyboard (UI)
- 🎬 Smooth Animations
  - Tile flip effects using framer-motion

---

## 🧠 Game Rules

- Guess the hidden 5-letter word
- You have 6 attempts
- After each guess:
  - 🟩 Correct letter & position
  - 🟨 Correct letter, wrong position
  - ⬜ Not in the word
